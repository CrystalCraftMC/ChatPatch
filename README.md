[![Build Status](https://travis-ci.org/CrystalCraftMC/ChatPatch.svg?branch=master)](https://travis-ci.org/CrystalCraftMC/ChatPatch)


## Usage

* **Users**:
    * `/chatpatch on`
    * `/chatpatch off`
* **Admins**:
    * `/chatpatch save` (saves config)
    * `/chatpatch defaults` (restores default config)
